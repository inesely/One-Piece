
# One Piece API Project

## Beschrijving
Dit is een interactieve single page application gemaakt met Vite.  
De applicatie gebruikt een publieke API om One Piece anime data op te halen en weer te geven.

## Functionaliteiten
- Data ophalen via een API
- Minimum 20 items tonen
- Zoekfunctie
- Sorteerfunctie
- Favorieten toevoegen
- Favorieten verwijderen
- Favorieten bewaren met localStorage
- Responsive card layout

## Gebruikte technologieën
- HTML
- CSS
- JavaScript
- Vite
- GitHub

## API
De data komt van de Jikan API.
